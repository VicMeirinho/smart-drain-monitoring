# Smart Drain Monitoring

Projeto desenvolvido no IFSC utilizando Arduino e sensores para monitoramento de nível de água em bueiros.

## Objetivo
Detectar quando o bueiro atinge níveis críticos de água, ajudando na prevenção de alagamentos urbanos.

## Tecnologias utilizadas
- Arduino
- Sensor de nível de água
- C/C++
- Eletrônica básica

## Funcionalidades
- Monitoramento do nível de água
- Detecção de estado crítico
- Sistema de alerta

## Contexto
Projeto acadêmico desenvolvido durante o curso técnico em Informática no IFSC.

## Melhorias futuras
- Envio de alertas em tempo real
- Integração com IoT
- Dashboard de monitoramento
